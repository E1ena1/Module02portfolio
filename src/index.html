Поддержка сайта

Быстро устраню любую проблему на вашем сайте

  • HTML5
    язык для структурирования и представления содержимого всемирной паутины.
  • CSS
    формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
  • JavaScript ES6+
    Язык программирования высокого уровня, использующийся для придания интерактивности веб-страницам
  • jQuery
    Набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML. Помогает получать доступ к элементу DOM
  • Bootstrap
    Свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны.
  • PHP
    C-подобный скриптовый язык[12] общего назначения, интенсивно применяемый для разработки веб-приложений.
  • Vue.js
    JavaScript-фреймворк с открытым исходным кодом для создания пользовательских интерфейсов.
  • WordPress
    свободно распространяемая система управления содержимым сайта с открытым исходным кодом; написана на PHP; сервер базы данных — MySQL.
